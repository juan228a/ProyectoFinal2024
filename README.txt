# 🛠️ Proyecto Final - Prest-AR (Alquiler de Herramientas)

Proyecto desarrollado como trabajo final de la Tecnicatura en Calidad y Desarrollo de Software.  
Consiste en una plataforma web para **alquiler de herramientas entre usuarios de Tucumán**, con funcionalidades completas para usuarios y administradores.

---

## 📌 Descripción

Prest-AR permite a los usuarios **registrarse, iniciar sesión, publicar herramientas para alquilar, reservar herramientas de otros usuarios y pactar la entrega**.  
Además, cuenta con un **panel de administración** desde donde un admin puede ver, editar u observar publicaciones, gestionar usuarios y moderar la plataforma.

---

## 🚀 Tecnologías utilizadas

- 💻 HTML, CSS, JavaScript
- 🐘 PHP (Back-End)
- 🧪 Katalon (testing funcional automatizado)
- ⚙️ Mocha + Chai (pruebas unitarias e integradoras)
- 🗃️ MySQL (base de datos)
- 📦 Git + GitHub (control de versiones)

---

## 💡 Funcionalidades principales

### 👤 Usuario:
- Registro e inicio de sesión
- Publicación de herramientas con imágenes
- Alquiler de herramientas de otros usuarios
- Acuerdo de lugar y hora para la entrega
- Perfil de usuario editable
- Visualización de reputación y reseñas

### 🛠️ Herramientas:
- Carga de imágenes
- Detalles como precio, categoría, días y horas disponibles
- Estado de disponibilidad y confirmación de alquiler

### 👨‍💼 Administrador:
- Panel para revisar publicaciones
- Edición o eliminación de herramientas
- Observaciones sobre herramientas cargadas
- Gestión de usuarios
